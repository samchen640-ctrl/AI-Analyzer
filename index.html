⚡️ SYSTEM WARNING // 系統連線異常

系統架構工程師的 CORS 排錯建議：

當前連線請求被瀏覽器的跨來源安全性沙盒政策 (CORS) 所阻擋。請使用以下方式排除：

  • 設定伺服器 CORS：請確保您的 AI 主機（如 LM Studio 或自行部署的 API 伺服器）啟動參數中已允許跨網域存取。例如加上：--cors=*。
  • 安裝瀏覽器擴充套件：最簡單的本地除錯方式是在瀏覽器中下載安裝並啟動 CORS Unblock 插件來暫時放行此請求。
  • 檢查 SSL/TLS 憑證：若連線網址為 https，請確認該網域的憑證是否有效且未過期，否則瀏覽器會出於安全直接拒絕存取。
